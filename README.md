# InTouchApp
InTouch is simply chat app based on Swift

### Features
- Can send and recieve messages
- Log-in/Register/Log-out functionality
- Firebase
- Swift 5

### Libraries used:
- SVProgressHUD
- Firebase
- ChameleonFramework
